### sunk

A library for interfacing with the Subsonic API.

### Example

TBA

### Currently supported

- Playlist controls
- Library scanning
- Retrieval (stream, download, HLS)

### Currently not supported

- Most browsing
- Searching
- Annotation
- Sharing
- Podcasts
- Jukebox
- Internet radio
- Chat
- User controls
- Bookmarks
- Anything to do with video

### Left to the implementor

- Playback (skipping, current playlist, etc.)
